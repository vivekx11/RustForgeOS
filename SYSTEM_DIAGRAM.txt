                              DATA FLOW EXAMPLES
═══════════════════════════════════════════════════════════════════════════════

1. HTTP REQUEST FLOW:
   Network Card → Interrupt → Kernel → Ethernet → IP → TCP → HTTP → Handler
   → Response → TCP → IP → Ethernet → Network Card

2. FORGESCRIPT EXECUTION:
   Source → Lexer → Parser → AST → Compiler → Bytecode → VM → Sandbox
   → Result

3. DATABASE TRANSACTION:
   Query → Parser → Planner → Transaction → B-Tree → WAL → Commit → Result

4. DISTRIBUTED WRITE:
   Client → Leader → Log → AppendEntries RPC → Followers → Majority
   → Commit → Apply → Response
