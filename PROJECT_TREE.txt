BUILD COMMANDS
═══════════════════════════════════════════════════════════════

🔨 Build:
   make build              # Build all crates
   make kernel             # Build bootable kernel
   cargo build --workspace # Direct cargo build

🧪 Test:
   make test               # Run all tests
   ./scripts/test-all.sh   # Comprehensive tests
   cargo test -p <crate>   # Test specific crate

🚀 Run:
   make run                # Run in QEMU (VGA)
   make qemu               # Run in QEMU (serial)
   make debug-qemu         # Run with GDB server

✨ Quality:
   make fmt                # Format code
   make clippy             # Lint code
   make doc                # Generate docs
